This program will create and compare disparity maps from two stereo images

------------------------------------
How to Compile the program:
------------------------------------

1. cd into the build directory "out\build\x64-Debug"\Skeleton.exe

2. pass the files paths to the two stereo images as arguments

------------------------------------
Operating the program once running:
------------------------------------

Use the left right arrow keys to browse the outputted data

Press [Escape] to stop the program.

------------------------------------
More info:
------------------------------------
See the run time display output for info on the disparity map generation.
